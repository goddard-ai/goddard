# Core System Specs

Defines the fundamental operations, rules, and runtime components shared by the SDK, app, and background automation.

## Core Specs

* `spec/core/architecture.md`: Stack choices and boundaries.
* `spec/core/runtime-loop.md`: Core execution loop, configuration constraints, and rate-limiting limits.
* `spec/core/data-flows.md`: Request and event sequences.
* `spec/core/inbox.md`: Daemon-local human-attention inbox semantics shared by SDK, app, and operational clients.
* `spec/core/session-titles.md`: Session title behavior, fallback guarantees, and optional title generation.
